Dự án web quản lý học tập + dự đoán điểm số của học sinh áp dụng mô hình Machine Learning

Công nghệ sử dụng:
- Front end: Reactjs
- Back end: Web API .net 8.0

Mô hình Machine Learning sử dụng trong dự án:
- Linear Regression
- RandomForest
- XGBoost
